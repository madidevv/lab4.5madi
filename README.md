# Лабораторная работа 4–5. Основы разработки интерфейсов мобильных приложений
# Цель работы

Изучение основ создания интерфейсов мобильных приложений: размещение элементов, настройка их свойств, создание прототипа интерфейса.

# Задачи

Ознакомиться с элементами интерфейса Android (EditText, Button, ImageView, FrameLayout, TextView).

Научиться размещать элементы на экране и менять их свойства через XML и графический редактор.

Разработать прототип интерфейса собственного приложения с элементами управления.

# Ход работы
# 1️⃣ Изучение элементов интерфейса

Создано главное окно приложения с базовыми элементами:

EditText – поле для ввода текста пользователем.

Button (Go) – кнопка для начала поиска изображения.

FrameLayout с ImageView – рамка с изображением, которое можно менять динамически.

Button Like/Dislike – кнопки оценки изображения; при нажатии показывается следующее изображение.

Размещена иконка приложения.

# 2️⃣ Практическое размещение и настройка

Добавлено текстовое поле <EditText> с атрибутами:

android:ems="10" – задаёт ширину поля для симметричного отображения текста.

android:hint="@string/edit_message" – подсказка по умолчанию.

Добавлены кнопки <Button> для поиска и оценки изображений.

В рамку <FrameLayout> помещён <ImageView> для отображения картинки.

# 3️⃣ Разработка прототипа

Кнопки Like/Dislike переключают изображения по кругу.

Добавлена строка <TextView> для отображения адреса сайта изображения.

Добавлена кнопка <Button> для перехода на сайт через Intent.ACTION_VIEW.

Логика работы приложения

Пользователь вводит текст в поле EditText.

Нажимает Go – можно реализовать поиск изображения по введённому тексту.

На экране отображается изображение.

Пользователь нажимает Like или Dislike, изображение меняется на следующее.

Адрес сайта текущего изображения отображается в TextView, и по кнопке можно перейти на сайт.
